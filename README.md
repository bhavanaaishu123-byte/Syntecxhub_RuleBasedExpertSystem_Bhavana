# Rule-Based Expert System

## Project Overview
This project is a Rule-Based Expert System developed using Python.

The system accepts user symptoms as input and applies if-then rules using forward chaining to infer conclusions.

It also logs inference steps to show the reasoning path.

## Features
- Rule-based reasoning
- Forward chaining inference
- Multi-step rule chaining
- Inference step logging
- Final diagnosis display

## Technologies Used
- Python

## Sample Input
fever, cough, body_pain

## Sample Output
viral_infection
